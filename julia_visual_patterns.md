# Padrões Visuais Consolidados - Projetos Jul.IA

## Análise Comparativa dos Três Projetos

### 1. Jul.IA - Automação de Petições
- **Tela de Login:** Design centralizado, minimalista, fundo branco
- **Botão primário:** Azul royal/índigo
- **Logo:** Ícone verde com símbolo de documento

### 2. Jul.IA - Envio de Intimações (Agenda)
- **Tela de Login:** Card centralizado com fundo branco sobre fundo azul claro/lavanda
- **Título:** "Jul.IA - Envio de Intimações"
- **Subtítulo:** "Sistema de Envio de Intimações"
- **Descrição:** "Envie intimações para seus clientes via e-mail ou WhatsApp de forma rápida e organizada"
- **Botão primário:** Azul royal com texto "Fazer Login" e seta
- **Fundo da página:** Azul claro/lavanda suave

### 3. Jul.IA - Automação de Procurações
- **Interface completa:** Dashboard funcional com sidebar e área de conteúdo
- **Sidebar:** Fundo branco com menu lateral colorido
- **Cores do menu:** Cada item tem uma cor de destaque diferente (laranja, roxo, verde, vermelho, azul)
- **Logo:** Ícone verde no topo da sidebar
- **Área principal:** Fundo branco com cards informativos
- **Cards de métricas:** Bordas suaves, ícones coloridos (azul, verde, roxo)
- **Cards de ações rápidas:** Fundo claro com ícones coloridos e setas de navegação

## Padrões Visuais Consolidados para Jul.IA WhatsApp

### Paleta de Cores
- **Primária:** Azul royal/índigo (#4F46E5 ou similar)
- **Secundária:** Verde (#10B981 ou similar) - usado no logo
- **Fundo principal:** Branco (#FFFFFF)
- **Fundo alternativo:** Azul claro/lavanda (#E0E7FF ou similar)
- **Texto primário:** Preto/cinza escuro (#1F2937)
- **Texto secundário:** Cinza médio (#6B7280)
- **Cores de destaque para menu/ícones:** Laranja, Roxo, Verde, Vermelho, Azul

### Tipografia
- **Família:** Sans-serif moderna (provavelmente Inter, Poppins ou similar)
- **Títulos principais:** Bold, tamanho grande
- **Subtítulos:** Medium/Semibold, tamanho médio
- **Texto descritivo:** Regular, tamanho padrão
- **Hierarquia clara:** Diferenciação por peso e tamanho

### Layout e Estrutura
- **Design responsivo:** Centralizado em telas de login
- **Cards:** Bordas arredondadas, sombras suaves
- **Espaçamento:** Generoso entre elementos
- **Sidebar (quando aplicável):** Menu lateral com ícones coloridos
- **Área de conteúdo:** Fundo branco, cards organizados em grid
- **Botões:** Arredondados, com hover states, cores vibrantes

### Componentes Comuns
- **Logo:** Ícone verde com fundo escuro (símbolo jurídico/documento)
- **Botões primários:** Azul royal, texto branco, arredondados
- **Cards informativos:** Fundo branco, bordas suaves, ícones coloridos
- **Ícones:** Estilo outline ou filled, cores variadas
- **Badge "Made with Manus":** Canto inferior direito

### Princípios de Design
- **Minimalismo:** Interface limpa e focada
- **Profissionalismo:** Cores sóbrias com toques de cor estratégicos
- **Usabilidade:** Hierarquia visual clara, CTAs evidentes
- **Consistência:** Padrões repetidos entre os projetos
- **Acessibilidade:** Contraste adequado, textos legíveis

## Aplicação no Jul.IA - Assistente de WhatsApp

O chatbot deve seguir estes padrões visuais em sua interface web de gerenciamento:

1. **Tela de boas-vindas/login:** Card centralizado sobre fundo azul claro/lavanda
2. **Dashboard:** Sidebar colorida + área principal com cards
3. **Cores:** Azul royal para ações primárias, verde para logo/sucesso
4. **Cards de métricas:** Leads qualificados, atendimentos humanos, conversões
5. **Cards de ações:** Configurar fluxo, visualizar conversas, exportar dados
6. **Tipografia:** Sans-serif moderna, hierarquia clara
7. **Ícones:** WhatsApp (verde), usuários, documentos, relatórios
